# Heritage Centre Interactive Homeland Map

## Project Overview
An interactive map of Red River Métis settlements and communities for the National Heritage Centre.

## Data Sources
- `RAW/CSV Data/Settlements_and_locations.csv` — Master dataset (228 locations with enriched metadata)
- `RAW/Photos/` — Provided photos (MET001-MET013.jpg)
- Wikimedia Commons — Open-source image sourcing (in progress)

## Map Features
- **228** location pins across the homeland
- **18** regional clusters
- **10** toggleable map layers
- **18** interconnected family networks
- **10 icon types** (fort, battle, church, settlement, etc.)
